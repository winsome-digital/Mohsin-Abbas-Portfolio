# Mohsin Abbas — Portfolio Website

A fully responsive, visually rich personal portfolio built from scratch with HTML5, CSS3, and Vanilla JavaScript. Designed to showcase my skills, experience, and projects as an AI Automation Specialist and Front-End Developer.

## ✨ Features

- 🌙 Dark / Light mode toggle
- 🖱️ Custom animated cursor with magnetic effect
- ✨ Interactive particle star field background
- 🃏 3D tilt project cards with modal popups
- ⌨️ Typewriter role animation in hero section
- 📊 Animated skill progress bars (scroll-triggered)
- 🖼️ Profile photo — add, change, or remove (persists via localStorage)
- 🖼️ Project screenshot slots — upload images per project
- 📱 Fully responsive with mobile hamburger menu
- 🔢 Animated counter stats
- 📬 Contact form with validation
- 🚀 Zero dependencies — pure HTML, CSS, JS
